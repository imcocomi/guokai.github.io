About Me

关于我

I am a PhD candidate at the University of Tokyo (UTokyo), focusing on research in binocular rivalry and its applications in user interface design and visual effects.

My research interests include human-computer interaction, visual perception, and innovative display technologies.

我是东京大学的博士候选人，专注于双眼竞争及其在用户界面设计和视觉效果中的应用研究。

我的研究兴趣包括人机交互、视觉感知和创新显示技术。

Research Topics

研究方向

Binocular Rivalry User Interface
双眼竞争用户界面
My research explores how binocular rivalry can be leveraged to create novel user interfaces that provide enhanced information display and interaction techniques. By presenting different stimuli to each eye, we can create unique perceptual experiences that have applications in augmented reality, accessibility, and information visualization.
我的研究探索了如何利用双眼竞争来创建提供增强信息显示和交互技术的新型用户界面。通过向每只眼睛呈现不同的刺激，我们可以创造独特的感知体验，这些体验在增强现实、可访问性和信息可视化方面有应用。
Project Website项目网站 GitHub Paper论文
Binocular Rivalry Visual Effects
双眼竞争视觉效果
This research direction focuses on creating visual effects that utilize dichoptic color differences to induce perceptual phenomena. By manipulating how colors are presented to each eye, we can create visual experiences that appear paranormal or impossible, with applications in entertainment, art, and perception studies.
这个研究方向专注于创建利用双眼色差诱导感知现象的视觉效果。通过操纵向每只眼睛呈现颜色的方式，我们可以创造看起来超自然或不可能的视觉体验，这些体验在娱乐、艺术和感知研究方面有应用。
