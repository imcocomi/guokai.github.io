研究项目

Binocular Rivalry User Interface
両眼視野闘争ユーザーインターフェース
双眼竞争用户界面
My research investigates how binocular rivalry can be utilized to design novel user interfaces that enhance visual flexibility. By presenting different UI elements to each eye, users can selectively focus their attention to either view the graphical interface or the underlying visual scene, even when the GUI is overlaid.
両眼に異なるUI要素を提示することで生じる視野闘争現象を活用し、柔軟な視覚体験を実現する新しいユーザーインターフェース設計に関する研究です。この手法により、GUIが背後のシーンに重なっている場合でも、ユーザーは注意を向ける対象を選択し、シーンまたはGUIのいずれかを明瞭に観察できます。
我的研究探索了如何利用双眼竞争现象，设计出能够提升视觉灵活性的新型用户界面。通过向左右眼分别呈现不同的界面元素，即使GUI叠加在背景场景上，用户也可以根据自身注意力的转移，自由选择观看完整的视觉场景或界面内容。

Binocular Rivalry Visual Effects
視野闘争エフェクト
双眼竞争视觉效果
This research direction focuses on creating visual effects that utilize dichoptic images to induce perceptual phenomena. By manipulating how colors, shapes, materials ,etc are presented to each eye, we can create visual experiences that appear paranormal or impossible, with applications in entertainment, art, and perception studies.
両眼に異なる色彩を提示する手法を用いて、知覚現象を誘発する視覚効果の創出に焦点を当てた研究です。色彩の提示方法を操作することで、超常現象のような視覚体験を生み出し、エンターテイメント、芸術、知覚研究などの分野に応用できます。
这个研究方向专注于创建利用双眼色差诱导感知现象的视觉效果。通过操纵向每只眼睛呈现颜色的方式，我们可以创造看起来超自然或不可能的视觉体验，这些体验在娱乐、艺术和感知研究方面有应用。
